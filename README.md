# GRATIS LOVE Website

This is the source code for the GRATIS LOVE website.

## Structure

- `index.html`: The main page of the website.
- `participants.html`: The page with the list of all participants.
- `recipes.html`: The page with the list of all recipes.
- `workshop.html`: The page with the list of all workshops.
- `PARTICIPANTS/`: This directory contains the individual pages for each participant.
- `img/`: This directory contains all the images used in the website.
- `style.css`: The main stylesheet for the website.
- `script.js`: The main script for the website.
- `works_data.js`: This file contains the data for the gallery. It is generated by the `generate_gallery.py` script.

## How to update the gallery

The gallery data is stored in the `works_data.js` file. This file is generated by the `generate_gallery.py` script. To update the gallery, you need to run this script.

1.  Make sure you have Python 3 installed.
2.  Run the script from the root of the project:

```
python3 generate_gallery.py
```

The script will scan the `img/works` directory and generate the `works_data.js` file.
